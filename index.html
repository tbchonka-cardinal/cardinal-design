Cardinality Design Reference

Cardinal Design System v1.0.0

An aged-ledger visual language of warm paper, dark ink, gold hairline rules, and Garamond set small, carried across every tool in the Cardinality suite so the ledgers all read as one hand.

Tokens

Every value below is a CSS custom property defined in tokens.css. Components consume these only; no raw hex or rgb anywhere in components.css.
